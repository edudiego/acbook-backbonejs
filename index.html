Lecture 1

In the first lecture, we cover the basics of Backbone.JS to give an overview of it's main features. We also cover how to install it to create a basic web application, and talk about the Backbone Views and Models.

Lecture 2

In the second lecture, we expand on the models and its remote data syncing capabilities. We then proceed to explain how Views and Models work together.

Lecture 3

In the third lecture, we go over the Router and History features, which allow us to use Backbone to create a fully realized Single Page Application. Next we cover the topics of Collections and explain how to use Collections and Views together.

Lecture 4

In the final lecture, we do a quick syntax review to refresh our knowledge and tackle common errors from first time users. Then we talk about the Social API that we use as an example of a backend endpoint for our app. Next, we go further on the Backbone.JS functionalities approaching its internal messaging features via event triggers and listeners, following up with our final considerations.
